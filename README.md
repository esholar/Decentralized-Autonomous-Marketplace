### Prerequisites
- Node.js (v18+)

- Truffle
- Web3.js
- IPFS
- Chainlink Oracles

### Setup Instructions

1. Clone Repository
```bash
git clone https://github.com/dadsm/digital-services-marketplace.git
cd digital-services-marketplace
```

2. Install Dependencies
```bash
npm install
```

3. Compile Contracts
```bash
npx hardhat compile
```

4. Run Local Blockchain
```bash
npx hardhat node
```

5. Deploy Contracts
```bash
npx hardhat run scripts/deploy.js --network localhost
```
